# 🧮 Zakat Calculator 2026

A simple and responsive **Zakat Calculator** based on the **2026 Nisab value (BDT)**.  
This tool helps users determine whether Zakat is obligatory and calculates **2.5%** of eligible wealth.

---

## 📌 Features

- ✅ 2026 fixed Nisab value (৳2,30,000)
- ✅ Automatic Zakat calculation (2.5%)
- ✅ Shows if Zakat is **not obligatory**
- ✅ Clean and modern UI
- ✅ Fully mobile responsive
- ✅ Beginner-friendly code (HTML, CSS, JS)

---

## 📊 Nisab (2026)

- 💰 **Nisab Amount:** ৳2,30,000 
- 📉 **Zakat Rate:** 2.5%  
- ⚠️ Applies only if wealth is above Nisab for 1 lunar year

---

## 🚀 How to Use

1. Enter your total Zakat-eligible wealth  
2. Click **"Calculate"**  
3. See:
   - If Zakat is required
   - The exact amount you need to pay
